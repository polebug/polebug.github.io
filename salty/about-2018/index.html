2018

气氛上我是一个不喜欢写什么年度总结的人，因为对我来说 2018/12/31 以及 2019/1/1 都是非常普通的两天。但鉴于我回外婆家过元旦干不了什么正经事，并且好像也很无聊的样子，就来尝试写写 2018 总结吧。

感觉最重要的是该回忆一下 2018 还留下了哪些坑没有填完（时间线是从最近开始往前）：

  • 1. 帅副给我推了清华大学的 os 公开课，并且强烈推荐刷完 8 个 lab (未填)
  • 2. 依旧是帅副推给我的课 Database System Implementation，数据库硬课同样也是我想刷的 (然鹅未填)
  • 3. C 语言小子集编译器从预编译到前后端 (目前只写了前后端，预编译未填)
  • 4. 好好学一波 Golang (才开始填)
  • 5. 好好学一波 postgreSQL (未填)

    • emmmmmm…气氛上还有一些未填的坑，现在有点忘了，前三个坑都是 12 月挖的，后面两个大概是历史遗留问题 :)

    2018 年比 2017 年学的东西相对比较不是那么杂乱，2017 年踩过机器学习的坑 (刷了一堆课，看了一堆书)，踩过 common lisp 函数式编程的坑，也好好刷过绩点，两学期的一等奖学金也拿了，后面发现都不是自己喜欢的样子。

    2018 年开始接触后端开发，开始尝试用 Django/Flask 这些 python web 框架做一些开发，一路学下来发现还蛮有趣的 (果然更喜欢低层一点的东西！XD)，然后 12 月开始找后端开发实习岗。虽然不是正常的秋招春招时间，但找实习的过程还挺顺利的，遇到的人也都非常 nice。

    其实自己也想好了，如果能找到自己喜欢的实习，就出去试试看，寻找一下新的刺激；如果找不到的话也不亏，在面试过程中能很清楚的明白自己还有哪些不足，滚回学校继续充充电，这样在学习起来更有一些目的性，也是蛮开心的。

    2019 嘛，希望能快点在内地看到夏目剧场版！(拜托了！🙏)

    以及希望自己工作再久，仍然对技术抱有热情~

    May the force be with you
