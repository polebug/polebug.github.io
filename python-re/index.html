Python 正则表达式 [不完全记录]

2018-02-03 阅读量次

由于最近在用 Django 写 return0.io 的服务端，开始碰到很多匹配问题。顺手在这里开个坑，记录所用到的正则表达式。
 

语法 说明 表达式实例 匹配到的字符串
. 匹配任意字符 (除”\n”以外)
\ 转义字符，能使后一个字符改变原来的意思 a\\b a\b
* 匹配前一个字符n次 (n>=0) abcd* abc 或者 abcddddd(…)
+ 匹配前一个字符n次 (n>=1) abcd+ abcd 或者 abcddddd(…)
? 匹配前一个字符0次或者1次 abcd? abc 或者 abcd
