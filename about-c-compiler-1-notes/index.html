词法分析器的作用: 扫描语句中的每个词法单元 (token)

词法单元

要想让编译器知道每句话的意思，那么首先要让它先认识每个词素。

我们知道对于 C 语言编程来说，每个单词符号都有它不同的含义。例如： 

int main(){
    return 0;
}

扫描得到的词素有

int
main
(
)
{
return
0
;
}

有的是关键字，像 int main return
有的是界符，像 ( ) { ; }
有的是常数，像 0
除此之外，程序中还可能会出现运算符 > < =，以及标识符等等。

总的来说，我们把 C 语言中的词素分为五类：
1.关键字，比如 int main using namespace if else for while 等，见文末附录
2.标识符，比如变量名，函数名，过程名，类名，常量名
3.常数
4.运算符 - + * / % ^ >> << 等
5.界符 ; , {} () 等
