这些错误会造成很麻烦的问题，需要数小时才能解决

图片来源： opensource.com

当你做错事时，承认错误并不是一件容易的事，但是犯错是任何学习过程中的一部分，无论是学习走路，还是学习一种新的编程语言，比如 Python。

为了让初学 Python 的程序员避免犯同样的错误，以下列出了我学习 Python 时犯的三种错误。这些错误要么是我长期以来犯的，要么是造成了需要几个小时解决的麻烦。

年轻人可要注意了，这些错误是会浪费一下午的！
