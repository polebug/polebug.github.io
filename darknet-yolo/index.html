报错: Segmentation fault: 11 并且在目录中未出现已识别的图片

解决办法:

2.1更改Makefile:
将opencv部分修改为如下代码:(分别修改COMMON/LDFLAGS)

ifeq ($(OPENCV), 1) 
COMMON+= -DOPENCV -I/usr/local/include
CFLAGS+= -DOPENCV
LDFLAGS+= `pkg-config --libs opencv` -L/usr/local/lib -lopencv_core -lopencv_highgui -lopencv_imgproc
COMMON+= `pkg-config --cflags opencv` 
endif

save
make

2.2用官网提供的另一种命令方式:
注意到一个很坑的地方,

The detect command is shorthand for a more general version of the command. It is equivalent to the command:
./darknet detector test cfg/coco.data cfg/yolo.cfg yolo.weights data/dog.jpg
